# First Test

## Goal

Prove the ticket dashboard works before connecting real services.

## Steps

1. Open the Streamlit app.
2. Confirm the sample ticket appears.
3. Run the FastAPI alert bridge.
4. Send a fake webhook payload.
5. Confirm a new ticket is added.
6. Later: connect Uptime Kuma webhook.
7. Later: connect NVIDIA summary.
8. Later: connect Discord notification.
